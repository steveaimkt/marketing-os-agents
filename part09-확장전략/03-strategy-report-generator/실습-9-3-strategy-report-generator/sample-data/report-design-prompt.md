# 클로드 디자인 복붙용 프롬프트 (임원 1페이지 시각 리포트) — 골격

> 용도 : 클로드코드가 2-3 에서 산출하는 `report-design-prompt.md` 의 골격 레퍼런스.
> 이 텍스트를 `report-content.md` 요지와 함께 **claude.ai(클로드 디자인)** 에 붙여넣으면
> 임원용 1페이지 시각 리포트(Artifact)가 생성된다. 아래는 2027 Q1 · 결로아(가상) 기준.

---

## 붙여넣을 프롬프트 (그대로 복사 · 수치는 report-content.md 에서 채움)

```
임원 보고용 "1페이지 분기 전략 리포트"를 시각적으로 만들어줘. 아래 사양을 정확히 지켜줘.

[브랜드/기간] 결로아(K-뷰티 D2C 토너) — 2027 Q1 분기 리포트
[용도] 임원 5분 브리핑용 1페이지 (스크롤 없이 핵심이 한눈에)

[레이아웃]
  · 상단 : 제목 + 기간 + 한 줄 총평
  · KPI 4카드 (가로 나열) : 매출 / 통합 ROAS / VIP LTV / NPS
       - 각 카드에 값 + 전분기 대비 증감(▲▼ %)
  · 중단 좌 : "분기 핵심 성과 3" 불릿
  · 중단 우 : "다음 분기 전략 3" 불릿
  · 하단 : 5장 목차(성과·고객·경쟁·전략)로 가는 안내 한 줄 (상세는 Notion)

[핵심 수치 (report-content.md 에서)]
  · 매출 : 지수/증감률로 표기 (외부공유본은 절대값 마스킹)
  · 통합 ROAS 2.71 (전분기 2.38 ▲)
  · VIP LTV 28.4만 (+11%)
  · GA4 세션 42.1만(+18%) · 전환율 2.34% · 신규 68%

[핵심 성과 3]
  1) Meta 전환 캠페인 ROAS 3.8 · 신규→재구매 전환 31%
  2) VoC 긍정 감성 72% · "저자극" 포지셔닝 반응 상승
  3) 네이버 브랜드 키워드 효율 개선

[다음 분기 전략 3]
  1) "저자극/리필" 트렌드 대응 신제품 캠페인
  2) 네이버 일반키워드 예산 재배분 (부진 → Meta 리타겟)
  3) At-Risk 세그먼트 윈백 자동화

[디자인 톤]
  · 임원 보고서 톤 : 절제·명료·신뢰 / 숫자가 주인공, 장식 최소
  · 라이트/다크 모두 가독 · KPI 증감은 색으로 (상승=강조색, 하락=경고색)
  · 한 화면에 다 들어오게(1페이지) · 인쇄/캡처해도 깨지지 않게
```

---

## 클로드코드가 프롬프트를 만들 때 지키는 규칙 (재현성)

1. **KPI 4개·성과 3·전략 3 를 프롬프트에 명시** — 임원 1p 의 고정 골격.
2. **수치는 report-content.md 집계값에서** 가져와 프롬프트에 박아넣기 (디자인이 지어내지 않게).
3. **외부공유본은 매출 절대값을 지수/증감률로 마스킹** 하도록 지시.
4. **1페이지·인쇄 안전** 요구 — 임원 브리핑 맥락(스크롤 없이 한눈).
5. 상세 5장은 시각 리포트에 넣지 말고 **Notion 다중 페이지로** — 1p 는 요약만.
