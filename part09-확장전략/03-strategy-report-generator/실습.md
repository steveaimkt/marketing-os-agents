# 실습 — strategy-report-generator

```bash
claude --agent strategy-report-generator "2026 Q2 분기 보고서"
# → 6개 에이전트 병렬 호출 → 데이터 통합 → 노션 5장 페이지 + HTML
# → 임원용 1페이지 요약은 별도 페이지
```

## 검증
- 5장 모두 1페이지 이상 본문 채워졌나
- 핵심 지표 4개가 일관되게 표시되나
- 다음 분기 전략 3가지가 데이터 기반인가
