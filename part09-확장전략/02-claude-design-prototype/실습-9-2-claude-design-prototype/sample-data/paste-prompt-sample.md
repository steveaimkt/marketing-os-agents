# 클로드 디자인 복붙용 프롬프트 (paste-prompt) — 예시 1 골격

> 용도 : 클로드코드가 2-3 에서 산출하는 `paste-prompt.md` 의 골격 레퍼런스.
> 이 텍스트를 **claude.ai(클로드 디자인)** 새 대화에 그대로 붙여넣으면 Artifact 프로토타입이 생성된다.
> 아래는 예시 1(신학기 모공 진단 미니앱) 기준. 실제로는 아이디어별로 클로드코드가 채워 만든다.

---

## 붙여넣을 프롬프트 (그대로 복사)

```
클릭 가능한 단일 파일 프로토타입을 만들어줘. 아래 사양을 정확히 지켜줘.

[제품] 결로아(K-뷰티 D2C 토너) — "신학기 모공 진단 미니 앱"
[목적] 사용자가 3문항에 답하면 점수로 토너 3종 중 1개를 추천하는 진단 흐름

[화면]
  ① 진단 화면 : 문항 1개씩 순차 표시 + 진행 표시(1/3)
  ② 결과 화면 : 추천 토너 카드 1개 + 한 줄 카피 + "다시 진단" 버튼

[상태]
  · step (0~2 문항 인덱스) · answers[] · score 합산

[입력]
  문항당 라디오 3택 (아래 스크립트)
  1. 세안 후 몇 시간 뒤 번들거리나요? → 바로(모공 +2) / 3~4시간(+1) / 거의 안 남(수분)
  2. 모공이 가장 신경 쓰이는 부위는? → T존·볼(모공 +2) / 부분적(+1) / 딱히 없음(민감)
  3. 자극·붉어짐이 있나요? → 자주(민감 +2) / 가끔(+1) / 없음(0)

[결과 분기]
  · 모공 신호 우세 → 포어 토너 V2 · "T존까지 조이는 모공 집중 케어"
  · 수분 신호 우세 → 워터 토너 · "속당김 없는 수분 결 채우기"
  · 민감 신호 우세 → 카밍 토너 · "자극 없이 진정부터, 저자극 포뮬러"

[디자인 톤]
  · 모바일 우선(375px) · 다크 모드 · 의존성 0 (외부 CSS/JS/폰트 없음, 단일 파일)
  · 결로아 톤 : 깔끔·정돈·신뢰 / 버튼은 크게, 1분 안에 클릭 검증 가능하게
  · 결과 화면은 추천 이유(선택 답변 요약)도 한 줄 보여주기
```

## 클로드코드가 프롬프트를 만들 때 지키는 규칙 (재현성)

1. **4컴포넌트를 프롬프트에 그대로 명시** — 화면/상태/입력/결과가 다 들어가야 결과가 흔들리지 않음.
2. **톤·제약을 마지막에 못 박기** — 375px·다크모드·의존성 0·단일 파일.
3. **카피·분기 규칙까지 포함** — 디자인이 문구를 지어내지 않도록 마케터가 카피를 준다.
4. **1분 검증 가능성** 요구 — 과하게 복잡한 인터랙션 금지(프로토타입은 검증용).
