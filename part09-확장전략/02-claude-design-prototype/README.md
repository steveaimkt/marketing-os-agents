# 클립 1-2. Claude Design — 웹·앱 프로토타입

[`agents/part9-expansion/claude-design-prototype.md`](../../../agents/part9-expansion/claude-design-prototype.md)

마케터의 아이디어 → **5분 안에 클릭 가능한 프로토타입** (HTML 단일 파일).

## 예시
- "모공 진단 미니 앱" (3문항 → 토너 추천)
- "21일 챌린지 인증 페이지"
- "신규 가입자 환영 흐름"

→ [`실습.md`](실습.md)

---

## 실습 스킬

이 실습은 **클로드코드=기획 / 클로드 디자인=생성** 으로 역할을 나눕니다. 클로드코드가 아이디어를 4컴포넌트로 분해해 `design-brief.md` + 복붙용 디자인 프롬프트를 만들고, 그 프롬프트를 **클로드 디자인(claude.ai Artifacts)** 에 붙여넣어 클릭 가능한 프로토타입을 생성하는 핸드오프 방식입니다.

촬영·실습은 전용 스킬로 진행합니다. **"9-2 실습 시작"** 이라고 입력하면
[실습-9-2-claude-design-prototype/SKILL.md](실습-9-2-claude-design-prototype/SKILL.md) 의 게이트 흐름 (점검 → 역할 분리 안내 → 아이디어 선택 → 4컴포넌트 분해 → design-brief.md + 프롬프트 산출 → 클로드 디자인 핸드오프) 이 시작됩니다.
예시 프롬프트·복붙 프롬프트 골격·완성 기준 레퍼런스는 [실습-9-2-claude-design-prototype/sample-data/](실습-9-2-claude-design-prototype/sample-data/) 에 포함되어 있습니다.
