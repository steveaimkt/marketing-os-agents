# 실습 — claude-design-prototype

```bash
claude --agent claude-design-prototype "$(cat <<'EOF'
신학기 모공 진단 미니 앱:
  - 3문항: 피부 타입(건성/지성/복합) / 모공 고민 부위 / 사용 단계
  - 결과 페이지: 본 토너 추천 + 21일 챌린지 신청
  - 모바일 우선, 다크 모드 자동
EOF
)"
# → outputs/prototypes/skin-quiz/index.html
# → 로컬 서버 자동 시작 + 디스코드에 URL
```

## 검증
- 모바일에서 3문항 흐름 1분 이내
- 결과 페이지 CTA 클릭 추적 가능

## 응용
- 마케팅 캘린더 캠페인을 자동으로 페이지화
- Figma 핸드오프 (디자이너 협업)
