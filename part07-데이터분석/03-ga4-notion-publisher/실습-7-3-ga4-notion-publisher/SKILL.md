> 7-2가 만든 **HTML 리포트를 노션 페이지로 변환하고 디스코드에 첨부·요약 발송**한다 (선택 시 사내 이메일까지).
> 핵심 학습 목표 : ① HTML → 노션 블록 변환 ② 산출물 3곳 동시 도달 ③ 7-1→7-2→7-3 파이프라인의 마지막 칸.
> ⛔ 자동 실행 금지. 단계마다 사용자 답을 받은 뒤에만 다음으로.


## 1단계 · 소개

```
📮 GA4 노션 발송 에이전트 실습을 시작합니다.

🎯 하는 일
  "리포트를 만들고 끝내지 않는다. 팀이 보는 곳(노션·디스코드)까지 자동으로 배달한다."
  7-2의 HTML 리포트를 받아 → 노션 페이지로 변환(KPI 카드·표) + 디스코드에 HTML 첨부·
  요약 embed + (선택) stakeholder 이메일 발송. 파이프라인의 마지막 배달 단계.

⏱ Before vs After
  · Before : HTML 열기 → 노션 수동 복붙 → 디스코드 업로드 → 메일 작성 = 약 20분/회
  · After  : 7-2 완료 후 자동 (또는 명령 1줄) → 노션·디스코드·(메일) 동시 도착 = 1분
```

🔌 연결 소스 (트리거 시 점검 · 실제 의존만)

```bash
claude mcp list 2>&1 | grep -i "notion"
claude mcp list 2>&1 | grep -iE "discord|gmail"
test -f "sample-data/ga4-weekly-report.html" && echo "샘플 HTML ✅"
```

```
  | # | 항목 | 용도 | 상태 |
  |---|---|---|---|

  | ② | Discord webhook          | HTML 첨부 + 요약 embed  | {✅ / ❌} |
  | ③ | Gmail MCP (선택)         | stakeholder 메일 발송   | {✅ / ❌} |
```

## 2단계 · 실행

### 2-0 · 사전 점검
1단계의 연결 소스 표에서 ❌ 인 항목을 먼저 해결한다.
샘플 모드면 내장 HTML(sample-data/ga4-weekly-report.html)을 입력으로 쓰고, 발송은 사용자 승인 뒤.

### 2-1 · 작동 구조
```
  [7-2 HTML 리포트 (또는 샘플 HTML)]
    → HTML 본문 → 노션 블록 구조로 변환
        · KPI 카드 4 · 인사이트 목록 · 채널 표 (차트는 이미지/요약으로)
    → "GA4 주간 리포트" 부모 페이지 아래 새 페이지 생성
    → 디스코드 : HTML 원본 첨부 + 요약 embed (#marketing-alerts)
        · 헤드라인(세션·전환·매출 WoW) + 핵심 신호 3 + 🔗 노션 링크 + 📎 HTML 첨부
    → (선택) stakeholder 이메일 발송  ✋ 사용자 승인
```
📄 에이전트 정의에서 새로 볼 부분 (`agents/part7-ga4/ga4-notion-publisher.md`):
  ① HTML → 노션 블록 변환 — 차트는 임베드/요약, 표는 노션 표로
  ② 산출물 3곳 동시 — 로컬(감사) + 노션(보관) + 디스코드(알림)
  ③ 결합 시나리오 — `/weekly-report` 로 7-1~7-3 자동 체이닝 (Part 10)

퀴즈: "이 에이전트가 stakeholder 메일을 자동으로 바로 쏘면?"
(답: 안 됨. 외부 발송은 ✋ 사용자 승인 게이트 — 발행 전 확인)

### 2-2 · 데이터 준비

```
📊 데이터 모드:

  [A] 샘플 데이터 모드 ⭐ 추천 · 촬영 기본
      sample-data/ga4-weekly-report.html 로드 (결로아 · W26 · 7-1/7-2와 동일 수치).
      KPI 카드 + 인사이트 + 채널 표가 담긴 컴팩트 HTML → 노션 변환 시연에 최적.
      노션은 촬영용 임시 페이지에 생성, 디스코드는 승인 뒤 발송.
      ※ 픽스처는 절대 수정하지 않는다 (재현성 보존).

  [B] 실데이터 모드
      7-2를 본인 GA4 시트로 먼저 실행 → 그 HTML 을 입력으로 사용.
      ⚠️ 촬영 시 속성 지표가 노션·디스코드에 노출됨을 확인하세요.
```

⏸ A/B 답 받고 2-3 으로.

### 2-3 · 실제 실행
```
▶ 에이전트 워크플로 단계별 실행:

  1. HTML 로드 (샘플 또는 7-2 산출물)
  2. HTML → 노션 블록 변환                  → (시연 하이라이트)
       · KPI 카드 4 : 세션 37,800 / 신규재방문 71·29 / 전환 1,168 / 매출 38.05M
       · 인사이트 3 : 유료 상승 · 모바일 저조 · 재방문 고전환
       · 채널 표 5행 (Organic~Referral)
  3. 노션 "GA4 주간 리포트" 페이지 생성       → 부모 페이지 하위 (마케팅 OS)
  4. 디스코드 발송  ✋ 승인 후                → #marketing-alerts
       · 요약 embed + 🔗 노션 링크 + 📎 HTML 첨부 (폰에서 바로 열림)
  5. (선택) stakeholder 이메일  ✋ 승인 후     → 사내 공유
```

⏸ 노션 페이지·디스코드 확인 후 "3단계" 라고 답해 주세요.
