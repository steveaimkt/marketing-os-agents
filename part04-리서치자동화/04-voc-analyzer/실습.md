# 클립 2-1 실습 — voc-analyzer

## Step 1. 소스 등록
```bash
claude "$(cat <<'EOF'
voc-analyzer 설정 노션 페이지 만들기:

소스 URL:
  - 자사몰 리뷰: https://brand.com/product/123/reviews
  - 올리브영 입점: https://oliveyoung.co.kr/store/goods/...
  - 인스타 멘션: @mybrand
  - X 검색: "토너 mybrand"

Gmail 라벨:
  - cs-inquiries (CS 문의)
EOF
)"
```

## Step 2. 첫 실행 (지난 7일)
```bash
claude --agent voc-analyzer "지난 7일 VoC 분석 실행"
# → 500건 이하 수집 → 분류 → 클러스터 → 시트·노션·디스코드 출력
```

## Step 3. 결과 확인
- 디스코드: TOP 3 페인 포인트 embed
- 노션: 주간 인사이트 페이지 (해결 액션 제안 포함)
- 구글 시트: 분류된 500행
