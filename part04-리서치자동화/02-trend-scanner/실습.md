# 클립 1-2 실습 — trend-scanner

## Step 1. 카테고리 설정
```bash
claude "$(cat <<'EOF'
trend-scanner 에이전트를 호출하기 전에, 본인 카테고리 설정 노션 페이지를 만들어줘:

페이지 제목: "trend-scanner 설정"
내용:
  - 카테고리: K-뷰티 D2C (토너·세럼 중심)
  - 핵심 키워드: 모공, 피지, 보습, 챌린지, 21일
  - 추적 해시태그: #모공케어 #토너추천 #21일챌린지 #케이뷰티
  - 제외어: 메이크업, 아이섀도 (관련성 낮음)
EOF
)"
```

## Step 2. 첫 실행
```bash
claude --agent trend-scanner "오늘 스캔"
# → 네이버+구글+X 트렌드 수집 → 노션 DB에 5~20행 추가 → 디스코드 TOP 3 알림
```

## Step 3. 결과 확인
- 노션 "주간 트렌드" DB → 행 확인
- 디스코드 → 급상승 TOP 3 embed

## Step 4. email-newsletter와 결합
```bash
# 일주일 후
claude --agent email-newsletter "테스트 발송"
# → trend-scanner가 쌓아둔 데이터를 자동으로 가져와 뉴스레터 작성
```

## 응용 과제
- 다른 카테고리 (예: 의류, 식품) 설정으로 변형
- WebSearch MCP를 더 활용해 뉴스도 수집
