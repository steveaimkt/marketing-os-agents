# 트렌드 스캔 설정 (Notion 설정 페이지 시딩용)

결로아 (Gyeolloa) 가상 브랜드 기준. 4-2 의 산출물 (Notion "주간 트렌드" DB)은
3-1 email-newsletter 의 입력이 된다 (파이프라인 체이닝).

## 카테고리

K-뷰티 D2C (토너·세럼 중심)

## 검색 소스 & 도구 (어디서 · 무엇으로 찾는가)

| 소스 채널 | 도구 (MCP) | 쿼리 / 스코프 | 결과 출처 표기 | 기본 |
|---|---|---|---|---|
| 네이버 (블로그·카페) | Firecrawl (search/scrape) | "키워드 site:naver.com" 검색·URL 스크랩 | "네이버" + 매체 | ON (키 필요) |
| 구글 / 일반 검색 | WebSearch | 도메인 제한 없음 | "구글" + 매체 도메인 | ON |
| 뉴스 | WebSearch | 키워드 + 연도/계절 | "뉴스" + 매체 | ON |
| SNS 해시태그 | WebSearch | #해시태그 검색 | "SNS" + 플랫폼 | ON |
| 특정 매체 심층 스크랩 | Firecrawl (search/scrape) | 지정 도메인·URL | 매체 도메인 | 선택 |
| 고정 URL 모니터 | WebFetch | 설정한 URL 목록 | 해당 URL | 선택 |

- 출처 표기 규칙: 모든 적재 행은 '출처' = (검색 채널) + (실제 매체 도메인) 을 기록한다.
  예) "네이버 · blog.naver.com" · "구글 · hwahae.co.kr" · "SNS · threads.com"
  → Notion DB '출처' 속성 + Discord embed 에 표기.
- 정직 고지(실측): WebSearch 크롤러는 naver.com 을 못 긁음(allowed_domains=naver.com → 400).
  → 네이버는 Firecrawl 경로 필수(.env FIRECRAWL 키). 미설치 시 구글 결과 중 naver 도메인만 출처 라벨로 폴백.
- 네이버 데이터랩·X·인스타 공식 API 는 전용 MCP 부재 → WebSearch 우회 또는 제외.
- 조절 knob: (a) 행별 ON/OFF (b) 쿼리 개수(비용↔커버리지) (c) allowed/blocked 도메인.

## 핵심 키워드 (관련성 채점 기준)

모공, 유분, 수분, 챌린지, 21일, 토너 패드, PHA, 저자극

## 추적 해시태그

#모공케어 #토너추천 #21일챌린지 #케이뷰티 #스킨케어루틴

## 제외어 (관련성 낮음 · 자동 드롭)

메이크업, 아이섀도, 립, 향수

## 관련성 점수 기준 (0~5)

- 5 : 핵심 키워드 2개 이상 + 토너/모공 직접 언급
- 3~4 : 핵심 키워드 1개 + 스킨케어 일반
- 0~2 : 제외어 포함 또는 카테고리 무관 → 드롭

## 급상승 판정

- 전주 대비 언급 빈도 1.5배 이상 또는 신규 등장 키워드
- TOP 3 만 디스코드 #marketing-research 로 발송

## 샘플 모드 검색 결과 픽스처 (실검색 대신 사용 가능)

샘플 모드에서는 아래 8건을 "이번 주 검색 결과"로 사용한다. 관련성 채점 시연용으로
고의로 점수대가 섞여 있다 (5점대 3건 · 3~4점대 3건 · 드롭 대상 2건).
각 건은 **출처(검색 채널 · 매체 도메인)** 가 부여돼 어디서 왔는지 추적된다:

1. "토너 패드 하나로 끝내는 아침 모공 루틴" : 출처 네이버·blog.naver.com : 모공+토너 패드 → 5점
2. "21일 챌린지 완주 후기 : 피부 결이 달라졌다" : 출처 SNS·instagram.com : 챌린지+21일 → 5점
3. "PHA vs AHA, 민감성 피부 각질 케어 비교" : 출처 구글·youtube.com : PHA+저자극 → 5점
4. "여름철 유분 잡는 스킨케어 순서" : 출처 네이버·blog.naver.com : 유분 → 4점
5. "올리브영 상반기 어워드 스킨케어 부문 결과" : 출처 뉴스·news.kr : 카테고리 일반 → 3점
6. "수부지를 위한 수분 크림 추천 7선" : 출처 구글·hwahae.co.kr : 수분 → 3점
7. "가을 신상 아이섀도 팔레트 스와치" : 출처 SNS·instagram.com : 제외어 → 드롭
8. "니치 향수 입문 가이드" : 출처 구글·youtube.com : 제외어 → 드롭

급상승 픽스처: "토너 패드" (전주 대비 2.3배) · "21일 챌린지" (1.8배) · "PHA" (신규 진입)
