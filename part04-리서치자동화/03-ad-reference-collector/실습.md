# 클립 1-3 실습 — ad-reference-collector

## Step 1. 시트 자동 생성
```bash
claude "$(cat <<'EOF'
Google Sheets에 "광고 레퍼런스 카탈로그" 새 시트 생성:
  컬럼: 수집일 / 광고주 / 카테고리 / 헤드라인 / 본문 발췌 / CTA / 이미지 URL / 관련성

탭 추가:
  - "이번주 인사이트" — 패턴 메모용
EOF
)"
```

## Step 2. 첫 실행
```bash
claude --agent ad-reference-collector "$(cat <<'EOF'
이번 주 실행:
  검색 키워드: 토너, 모공, 21일 챌린지, 보습, 나이아신아마이드
  국가: 한국
  기간: 최근 14일
  최소 관련성: 3점
EOF
)"
# → 30~80행이 시트에 추가됨 + 노션에 인사이트 페이지 생성
```

## Step 3. 검색·필터
시트에서 컬럼 필터로:
- 관련성 ≥ 4 → 본 광고 만들 때 직접 참고
- CTA 컬럼에서 자주 등장하는 문구 추출
