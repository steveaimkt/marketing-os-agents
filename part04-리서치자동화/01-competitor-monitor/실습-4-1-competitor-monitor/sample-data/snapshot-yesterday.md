# 전일 스냅샷 픽스처 (diff 시연용)

> 용도: 4-1 첫 실행은 원래 "초기 스냅샷 저장"만 하고 끝난다. 촬영에서 diff 장면을
> 바로 보여주기 위해, 이 픽스처를 "어제 저장된 스냅샷"으로 Notion 아카이브에 선시딩한다.
> 오늘자 Firecrawl 실스크랩 (또는 아래 "오늘 변화 시나리오") 와 비교하면 변경 3건이 검출된다.

## MEDICUBE (medicube.co.kr) · 어제 메인 페이지 요약

- 히어로 배너: "제로 모공 패드 2.0 · 단독 기획전 30% 할인" (종료일: 어제까지)
- 베스트 상품 1위: 제로 모공 패드 2.0 / 29,000원
- 공지: 여름 세일 예고 배너 없음

## TORRIDEN (torriden.com) · 어제 메인 페이지 요약

- 히어로 배너: "다이브인 저분자 히알루론산 세럼 · 수분 광채"
- 베스트 상품 1위: 다이브인 세럼 50ml / 22,000원
- 카피: "속수분 채움"

## NUMBUZIN (numbuzin.com) · 어제 메인 페이지 요약

- 히어로 배너: "3번 세럼 · 물광 톤업"
- 베스트 상품 1위: 3번 세럼 / 24,900원
- 이벤트: 리뷰 이벤트 진행 중 (포토 리뷰 적립금 2,000원)

---

## 오늘 변화 시나리오 (샘플 모드에서 "오늘 스냅샷"으로 사용)

샘플 모드는 Firecrawl 실스크랩 대신 아래를 오늘 스냅샷으로 사용해 diff 를 재현한다:

- **MEDICUBE**: 히어로 배너가 "여름 모공 빅세일 최대 50%" 로 교체 → 분류: 이벤트 + 가격
- **TORRIDEN**: 변화 없음 → diff 0건 (변화 없는 경쟁사도 있다는 것을 시연)
- **NUMBUZIN**: 신제품 "5번 토너 패드" 가 베스트 1위로 등장 / 가격 26,900원 → 분류: 신제품

## 기대 인사이트 (에이전트가 뽑아야 할 3건 예시)

1. MEDICUBE 가 여름 세일을 모공 키워드로 선점 : 결로아 21일 챌린지 시기와 겹침 → 차별화 메시지 필요
2. NUMBUZIN 의 패드형 신제품 진입 : 토너 패드 포맷 경쟁 심화 신호
3. TORRIDEN 은 수분 포지셔닝 유지 : 모공 카테고리 직접 경쟁자는 아님
