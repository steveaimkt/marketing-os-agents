# 클립 1-1 실습 — competitor-monitor

## Step 1. 경쟁사 DB 세팅
```bash
claude "$(cat <<'EOF'
Notion에 DB "경쟁사" 생성:
  컬럼: 브랜드명(Title), 도메인(URL), 인스타핸들(Text), 광고라이브러리URL(URL), 카테고리(Select)

3개 행 추가 (본인 카테고리에 맞게):
1. MEDICUBE / https://medicube.co.kr / @medicube_kr / ... / 토너
2. TORRIDEN / https://torriden.com / @torriden_official / ... / 세럼
3. NUMBUZIN / https://numbuzin.com / @numbuzin_kr / ... / 토너
EOF
)"
```

## Step 2. 첫 스캔
```bash
claude --agent competitor-monitor "오늘 스캔 실행"
# → 3개 도메인 스크랩 → 노션 스냅샷 저장 → 디스코드 알림
# (첫 실행은 비교 대상이 없으므로 "초기 스냅샷 저장" 메시지)
```

## Step 3. 다음 날 비교 결과 보기
하루 후 다시 실행하면 변경점 보고서가 나옴. 즉시 보고 싶으면 도메인 1개 수동 수정 후:
```bash
claude --agent competitor-monitor "강제 diff 모드. 어제 스냅샷 = 임의 1주 전"
```

## Step 4. 스케줄 등록 (Part 10에서 자세히)
```bash
# launchd plist 미리보기
cat <<'EOF'
<key>ProgramArguments</key>
<array>
  <string>/usr/local/bin/claude</string>
  <string>--agent</string>
  <string>competitor-monitor</string>
</array>
<key>StartCalendarInterval</key>
<dict><key>Hour</key><integer>9</integer></dict>
EOF
```

## 응용 과제
경쟁사 광고 라이브러리 URL을 노션 DB에 추가하고, Meta 광고도 함께 추적
