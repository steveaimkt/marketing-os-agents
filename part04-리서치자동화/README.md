# Part 4. 리서치 자동화 — 매주 5시간 사라지게 하는 5개 리서치 에이전트

> 이 Part에서 배우는 것: **마케터의 시간 도둑 1위인 리서치 업무를 5개 에이전트로 전부 자동화**.

## 왜 필요한가

리서치 업무는 매주 평균 5시간 (1인 마케터 기준):
- 경쟁사 모니터링: 1.5시간
- 트렌드 스캔: 1시간
- 광고 레퍼런스 수집: 1시간
- 고객 VoC 분석: 1시간
- SEO 키워드 리서치: 0.5시간

이걸 5개 에이전트가 매일/매주 자동 실행 → 사용자는 결과만 검토.

## 5개 에이전트

| # | 이름 | 무엇을 하나 | 트리거 |
|---|---|---|---|
| 1 | **competitor-monitor** | 경쟁사 사이트/SNS 변경점 감지 | 매일 09:00 |
| 2 | **trend-scanner** | 네이버·구글 키워드 트렌드 추출 | 매일 08:00 |
| 3 | **ad-reference-collector** | 국내외 광고 소재 수집 | 매주 월 |
| 4 | **voc-analyzer** | 리뷰·댓글 자동 분류 | 매주 화 |
| 5 | **seo-keyword-research** | 검색량·경쟁도 분석 | 키워드 입력 시 |

## 공통 출력 표준

모든 리서치 에이전트는 다음 3곳에 결과를 보낸다:
1. **Discord embed** — 짧은 요약 + 핵심 변경점 3개
2. **Notion DB** — 전체 데이터 + 메모 가능
3. **(선택) Google Sheets** — 시계열 추적용
