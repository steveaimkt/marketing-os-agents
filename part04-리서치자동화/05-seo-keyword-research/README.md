# 클립 2-2. SEO 키워드 리서치 에이전트

## 한 줄 요약
시드 키워드 1개 → **연관 키워드 50~100개로 확장 → 검색량·경쟁도 분석 → 우선순위 시트** 자동 생성.

## 에이전트
[`agents/part4-research/seo-keyword-research.md`](../../../agents/part4-research/seo-keyword-research.md)

## 점수 공식
```
priority = (월간 검색량 × 전환 가능성) / 경쟁도
```

검색량만 보지 않고 **경쟁도 + 카테고리 적합성**을 함께 보는 것이 핵심.
