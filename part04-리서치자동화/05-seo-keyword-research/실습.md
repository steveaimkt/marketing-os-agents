# 클립 2-2 실습 — seo-keyword-research

## Step 1. 시드 키워드로 실행
```bash
claude --agent seo-keyword-research "$(cat <<'EOF'
seed_keyword: "토너 모공"
target_market: "한국"
content_format: "blog"
EOF
)"
# → 50~100개 확장 → TOP 20 시트 + TOP 10 디스코드 embed
```
