# Part 4 / 4-5 · SEO 키워드 리서치 실습 : 소개 → 실행 → 활용

> 시드 키워드 1개를 **연관 30개로 확장하고 우선순위 공식으로 TOP 20 을 추리는** 에이전트를 가동한다.
> 핵심 학습 목표 : ① 우선순위 공식 (검색량 × 전환 잠재력 / 경쟁도) ② 의도 기반 제외 ③ 키워드 → 콘텐츠 변환.
> ⛔ 자동 실행 금지. 각 단계마다 사용자 답을 받은 뒤에만 다음으로.

## 진행 흐름 (3 단계)

```
1단계  소개  — 무엇을 하는 에이전트인가 + 연결 소스          ⏸ 답 대기
2단계  실행  — 사전 점검 · 작동 구조 · 데이터 준비 · 실제 실행  ⏸ 답 대기
3단계  활용  — 내 업무 적용법 + 산출물 + Part 4 마무리         ⏸ 선택
```

## 1단계 · 소개

```
🔑 SEO 키워드 리서치 에이전트를 소개합니다.

핵심 한 줄 : "감으로 고르지 않는다. 검색량 × 전환 잠재력 / 경쟁도 공식으로 줄 세운다."

⏱ Before vs After
  · Before : 키워드 도구 돌리고 엑셀 정리 + 어떤 글 쓸지 또 고민 = 약 2시간/회
  · After  : 시드 1개 → TOP 20 + 글감 5건까지 자동 = 약 5분/회

🔌 연결 소스 (이 에이전트가 실제로 의존하는 것만):
   seo-keyword-research
     ├─ WebSearch / Firecrawl  → 연관 키워드 확장 · 검색량 추정 보조
     └─ Google Sheets MCP      → TOP 20 시트 적재
     └─ Discord webhook        → TOP 10 발송
   참고 : 네이버 검색광고·구글 키워드 플래너 API 는 미연결이 기본.
          실데이터에서도 검색량은 WebSearch 추정치로 동작 (정밀 검색량은 별도 발급 안내).

💡 핵심 포인트
   ① 우선순위 공식이 프롬프트에 수식으로 명시 → 재현 가능한 판단 기준
   ② 검색량이 높아도 "의도"가 다르면 버린다 (시술 가격 · 무관 카테고리 · 규제 경계)
   ③ 산출 TOP 20 은 Part 5-4 콘텐츠 캘린더의 "이번 주 글감" 입력이 된다

👉 2단계(실행)로 넘어갈까요?
```

📐 **스킬? 에이전트?** → 도구(WebSearch·Sheets) + 트리거(회차 실행) + 산출(Discord)이 있으니 🤖 **에이전트**. 만드는 법은 3-1의 **에이전트 5요소** 골격 그대로.

### 2-1 · 작동 구조

```
🔄 에이전트 워크플로
  [시드 키워드 : "토너 모공"]
    → 연관 키워드 30개 확장 (LSI 포함)
    → 지표 수집 : 월간 검색량 · 경쟁도(0~1) · 전환 잠재력(1~5)
    → 우선순위 점수 = (월간 검색량 × 전환 잠재력) / 경쟁도 / 1000 (반올림)
    → 의도 기반 제외 (시술 의도 · 카테고리 무관 · 규제 경계)
    → TOP 20 시트 적재 + TOP 10 디스코드 + TOP 5 콘텐츠 제안

📄 에이전트 정의에서 새로 볼 부분 (agents/part4-research/seo-keyword-research.md):
   ① 공식이 프롬프트에 수식으로 명시됨 (결정적 점수화)
   ② 제외 규칙 : 검색량이 높아도 의도가 다르면 버린다
   ③ 규제 인지 : "모공 축소" 같은 표현은 화장품법 경계 → 콘텐츠 제안에 주석

❓ 퀴즈 : 검색량 27,300 의 "블랙헤드 없애는 법"이 TOP 5 콘텐츠 제안에 못 든 이유는?
   (답 : 경쟁도 0.81 + 전환 잠재력 2 → 점수 67 로 하락)
```

### 2-2 · 데이터 준비

```
📊 데이터 모드:

  [A] 샘플 데이터 모드 ⭐ 추천 · 촬영 기본
      시드 "토너 모공" + 키워드 30개 지표 픽스처. 점수화·제외 판단이 결정적으로 재현.
      이 TOP 20 은 Part 5-4 콘텐츠 캘린더 실습의 입력과 동일 수치.

  [B] 실데이터 모드
      본인 시드 키워드 1개 입력 → WebSearch 로 확장 + 추정 지표 (정밀도 낮음 주의).
```

`sample-data/seo-seed.json` + `sample-data/keyword-metrics.json` 로드 (⚠️ 픽스처 수정 금지):

1. 시드·시장·포맷·공식 확인 (시드 "토너 모공" · KR · blog · 공식 = (검색량 × 전환) / 경쟁도 / 1000)
2. Google Sheets **"SEO 키워드 TOP 20"** 생성 (6컬럼 : 키워드 · 월간 검색량 · 경쟁도 · 전환 잠재력 · 우선순위 점수 · 추천 콘텐츠)
3. 지표 픽스처 30개 중 미리보기 5건

[B] 실데이터 모드면: 시드 키워드·업종·콘텐츠 포맷 질문.

### 2-3 · 실제 실행

```
▶ 워크플로 단계별 실행:

  1. 시드 확장                            → 30개 키워드 목록
  2. 지표 결합 + 점수 계산                → 점수순 정렬 표 (시연 하이라이트)
       1위 "모공 토너 추천" 178점 : 검색량 22,100 × 전환 5 / 경쟁 0.62
  3. 의도 기반 제외 5건                   → 제외 사유 표시
       (피부과 시술 가격 · 화장솜 추천 · 메이크업 픽서 · 토너 영어로 · 여드름[규제 경계])
  4. Google Sheets TOP 20 적재
  5. TOP 5 콘텐츠 제안                    → 블로그 4 + IG 캐러셀 1 (각 제안의 근거 한 줄)
  6. Discord TOP 10 embed (점수 + 추천 포맷)
```

⏸ 각 단계 결과 확인 후 3단계로.
