# 3-3 실습 입력 데이터 : newsletter-writing 스킬 직접 호출용

결로아 (Gyeolloa) 가상 브랜드 세계관. 3-1 의 weekly-trends.json 과 동일한 출처를 사용해
파이프라인 일관성을 유지한다.

## 자료 3건 (sources)

1. **올영세일에서 토너 카테고리 매출 전년比 42% 성장**
   - URL: https://example-beautynews.kr/oliveyoung-toner-2026
   - 요약: 토너가 스킨케어 1위. '모공 수렴' 키워드 제품이 상위 10개 중 6개. 어성초·PHA 성분 강세.

2. **인스타그램 릴스 'NN일 챌린지' 해시태그 사용량 3개월 새 2.1배**
   - URL: https://example-snstrend.kr/challenge-hashtag-2026
   - 요약: 기간형 챌린지 해시태그 급증. 피부 루틴 인증 UGC 참여율 4.8%로 일반 포스트의 2배.

3. **성분 앱 리뷰 분석: 토너 구매 결정 1순위는 '수렴감'보다 '저자극'**
   - URL: https://example-reviewdata.kr/toner-review-insight-2026
   - 요약: 리뷰 12만 건 분석. 재구매 사유 1위 저자극(31%), 2위 모공 케어 체감(24%).

## brand_voice

- 톤: 친근하지만 신뢰감 있게 (전문 용어는 풀어서)
- 핵심 어휘: 케어, 챌린지, 루틴, 가이드, 저자극, 결
- 금기어: 미백, 주름 개선, 치료, 의학적 효능 단정 (화장품법 위반 소지)

## audience (페르소나)

- 25~34세 여성, 모공·유분 고민
- 성분을 따져 보고 사는 편, 過대광고에 피로감
- 출퇴근 시간에 모바일로 뉴스레터를 읽음 (스캔 가능해야 함)

## CTA

- 문구: "21일 모공 챌린지, 이번 주 시작입니다"
- URL: https://gyeolloa.kr/challenge-21d

## 기대 출력 (스킬 출력 스키마 검증용)

- 본문 약 800자 (자료 3건이 자연스럽게 엮인 내러티브)
- 제목 5종 (각기 다른 앵글: 긴급성 / 호기심 / 권위 / 사회적 증거 / 혜택)
- 1줄 요약 (프리헤더 60자 이내)
- CTA 1문장
