# 클립 1-3 실습 — newsletter-writing 스킬

## Step 1. SKILL.md 구조 확인
```bash
claude "skills/newsletter-writing/SKILL.md를 읽고 입력·출력 스키마를 정리"
```

## Step 2. 직접 호출 테스트
```bash
claude "$(cat <<'EOF'
Skill newsletter-writing 호출:

sources:
  - title: "K-뷰티 토너 시장 성장률 16%"
    summary: "2026 Q1 한국 토너 카테고리 성장률이 전년 대비 16% 증가."
    url: "https://example.com/article-1"
  - title: "나이아신아마이드 10% 효능 연구"
    summary: "서울대 피부과학팀 연구 결과 모공 -38%."
    url: "https://example.com/article-2"
  - title: "21일 챌린지 콘텐츠 트렌드"
    summary: "MZ세대 SNS 챌린지 참여율 평균 3.2배 증가."
    url: "https://example.com/article-3"

brand_voice:
  tone: "친근하지만 신뢰감 있는"
  vocabulary: ["케어", "챌린지", "가이드"]
  forbidden: ["미백", "주름개선"]

audience:
  - persona: "20-30대 여성"
  - core_pain: "모공·피지"

cta:
  text: "21일 챌린지 신청"
  url: "https://brand.com/challenge"
EOF
)"
# → 본문 800자 + 제목 5종 + preheader가 응답
```

## Step 3. 다른 에이전트에서 호출

`email-newsletter` 에이전트가 이미 이 스킬을 의존하도록 작성되어 있음. 호출만으로 결합 검증:

```bash
claude --agent email-newsletter "테스트 모드. 본인에게만 발송"
# → 내부적으로 newsletter-writing 스킬이 자동 호출됨
```
