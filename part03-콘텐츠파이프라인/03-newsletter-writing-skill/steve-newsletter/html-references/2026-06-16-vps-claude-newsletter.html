❶ 왜 다들 맥미니부터 고민할까

클로드 코드를 제대로 써본 분이라면 한 번쯤 부딪히는 순간이 있습니다. 노트북을 닫으면 작업이 멈춥니다. 책상을 떠나면 끝이에요.

외부에서 휴대폰으로 호출하고 싶은데, 그러려면 컴퓨터 한 대를 24시간 켜둬야 합니다. 그래서 많은 분들이 같은 결론에 도착해요. "그냥 맥미니 한 대 사서 항상 켜둘까?"

문제는 가격입니다. 맥미니 M4 기본 모델이 119만 원. 최근 80만 원대에서 30만 원 넘게 올랐습니다. 거기에 책상 한 자리, 외부 접속 별도 세팅, 고장 나면 A/S까지. 한 번 사면 끝이 아니죠. 그래서 제가 찾은 답이 VPS입니다.

❷ 월 1.3만 원이 119만 원을 이기는 계산법

VPS는 한마디로 클라우드에 있는 컴퓨터 한 대를 빌리는 것입니다. 24시간 켜져 있고, 어디서든 접속되고, 월 사용료만 내면 끝. 호텔 객실을 장기 임대한다고 생각하면 쉽습니다. 책상을 살 필요도, 청소할 필요도, 직접 고칠 필요도 없어요.

단순히 싸기만 한 게 아닙니다. 하드웨어를 안 사니 공간도 안 잡고, 고장 걱정도 없고, 어디서든 같은 환경에 접속할 수 있어요. 오늘 만드는 시스템 흐름은 4단계입니다 — VPS 위에 클로드 코드를 올리고 → 채널 플러그인을 깔고 → 디스코드 봇을 연결하면 → 디스코드 메시지를 VPS의 클로드 코드가 받아서 답을 줍니다.

사전 준비는 둘. ① Claude Pro 또는 Max (무료 플랜은 작동 안 함), ② 호스팅어 VPS KVM 2 (쿠폰 TRUMAN 으로 10% 추가 할인).

출처 · 마케팅 트루먼쇼

❸ 5분 설치 + 7단계 봇 연결

호스팅어의 가장 큰 장점은 브라우저 터미널입니다. SSH 클라이언트를 따로 깔 필요 없이 대시보드 버튼 하나로 터미널이 열려요. VPS가 처음인 분에게 이 기능 하나가 진입 장벽을 통째로 없애줍니다. 핵심만 추리면 이렇습니다.

⚙️ 클로드 코드 설치 — 5단계 (약 5분)
  • 공식 docs에서 curl 한 줄 복사 (2026.1부터 Node·npm 불필요)
  • 브라우저 터미널에 붙여넣고 10초 대기
  • 안내되는 PATH export 한 줄 추가
  • claude 실행 → 다크모드 · 로그인 1번(Pro/Max) → Authorize·Copy·붙여넣기
  • claude doctor 진단 + 폴더 신뢰 → 끝 (자격증명 저장됨)
🤖 디스코드 봇 연결 — 7단계
  • Discord Developer Portal에서 앱 생성 + Bot 토큰 발급 (외부 공유 금지)
  • Message Content Intent 활성화 → OAuth2로 봇을 내 서버에 초대
  • 클로드 코드에서 /plugin install discord@claude-plugins-official
  • /discord:configure <token> 으로 토큰 등록
  • ⚠️ claude --channels … 옵션으로 반드시 재시작 — 여기를 빼먹어 "왜 응답을 안 하지?"로 30분 날리는 분이 정말 많습니다
  • 봇에게 DM → 받은 페어링 코드를 /discord:access pair <code>
  • /discord:access policy allowlist 로 본인만 호출 가능하게 잠금

세팅이 끝났다면 봇에게 "작업 가능한 기능 알려줘"라고 보내보세요. 파일 읽기·명령 실행·도커 관리·깃허브 연결까지 다 된다고 답이 옵니다. 휴대폰 한 대로 24시간 작동하는 비서가 진짜로 완성되는 순간이에요.

추천 대상은 명확합니다. 클로드 코드를 24시간 돌리고 싶지만 맥미니가 부담인 분, 노트북을 닫으면 작업이 멈춰 불편했던 분, 외부에서 휴대폰으로 호출하고 싶은 분, SSH 세팅이 막막했던 분이라면 — 오늘 구조가 가장 빠른 답입니다.

📌 핵심 요약

Step 1. 호스팅어 VPS KVM 2 준비(쿠폰 TRUMAN 10% 할인) → 브라우저 터미널에서 curl 한 줄로 클로드 코드 설치. Node·npm 사전 설치 불필요.

Step 2. 디스코드 봇 생성 → 채널 플러그인 설치 → --channels 옵션으로 반드시 재시작. 봇 무응답 99%의 원인이 이 한 단계.

Step 3. 페어링 코드로 연결 → allowlist로 본인 외 접근 잠금. 비용은 맥미니의 절반 이하, 책상 자리 0, 휴대폰 하나로 24시간 비서 완성.
