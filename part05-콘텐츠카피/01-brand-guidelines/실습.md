# 클립 1-1 실습 — brand-guidelines

## Step 1. 본 브랜드 자산 URL 수집
```bash
claude "$(cat <<'EOF'
다음 URL 30개를 분석 대상으로 brand-guidelines 에이전트에 전달:

웹사이트:
  - https://mybrand.com (메인)
  - https://mybrand.com/about
  - https://mybrand.com/products/toner

인스타 인기 게시물 TOP 10:
  - https://instagram.com/p/...

블로그 인기 글 TOP 10:
  - https://mybrand.com/blog/...
EOF
)"
```

## Step 2. 실행
```bash
claude --agent brand-guidelines "본 브랜드 분석 실행"
# → 30개 자산 스크랩 → 5가지 항목 추출 → 노션 "Brand Voice" 페이지 생성
```

## Step 3. 검증
- 노션 페이지에서:
  - [ ] 톤 형용사 3개
  - [ ] 핵심 어휘 TOP 20
  - [ ] 금기어 카테고리
  - [ ] 페르소나 2명
- 마음에 안 들면 직접 편집 후 다시 사용 (`brand-voice` 스킬이 항상 최신 페이지를 읽음)

## 응용 과제
- 분기에 1번 자동 갱신 (`schedule: 0 17 1 */3 *` 사용 — Part 10)
