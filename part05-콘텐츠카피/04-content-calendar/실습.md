# 클립 2-1 실습 — content-calendar

```bash
claude --agent content-calendar "다음 주 콘텐츠 계획 (2026-W21)"
# → 노션 캘린더 DB에 30~50건 Draft 추가
# → 디스코드에 요약 embed
```

## 다음 단계 결합
```bash
# 월요일에 큐잉
claude --agent content-publisher "이번 주 Draft 큐잉"
# → 디스코드 #marketing-approvals 에 승인 요청
```
