# Part 5. 콘텐츠·카피 — 1주일치 콘텐츠를 1시간에 끝내는 5개 에이전트

> 이 Part에서 배우는 것: 카피 작성 + 검수 전체를 5개 에이전트와 2개 스킬로 자동화. 마케터의 글쓰기 시간을 90% 압축.

## 5개 에이전트 + 2개 스킬

| # | 이름 | 역할 |
|---|---|---|
| 1 | **brand-guidelines** | 본 브랜드 보이스·톤·페르소나 자동 추출 → 노션 가이드 |
| 2 | **ad-copy-ab** | 1회 입력 → 10가지 카피 변형 → 시트 정리 |
| 3 | **landing-copy** | 상품·리뷰·경쟁사 종합 → 노션 상세페이지 |
| 4 | **content-calendar** | 주간 콘텐츠 일정 자동 기획 → 노션 캘린더 |
| 5 | **quality-reviewer-6axis** | 6축 점수화 → 디스코드 피드백 |
| S1 | **brand-voice (스킬)** | 브랜드 톤 적용 패턴 |
| S2 | **ad-copy-ab (스킬)** | 광고 카피 변형 생성 패턴 |

## 6축 평가 모델 (이 Part의 핵심)

`quality-reviewer-6axis`가 모든 카피를 6축으로 점수:

1. **명확성** — 첫 문장에 핵심?
2. **타겟 적합성** — 페르소나와 맞나?
3. **차별성** — 경쟁사와 다른가?
4. **CTA 강도** — 행동을 유도하는가?
5. **브랜드 일관성** — `brand-voice` 가이드와 맞나?
6. **규제 안전성** — 식약처·표시광고법 위반?

각 축 0~5점, 평균 3.5 미만은 자동 재작성 권장.
