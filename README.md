# 마케터를 위한 클로드코드 : Part 3~10 에이전트 & 스킬

마케팅 자동화 에이전트와 실습 스킬 모음입니다. 클립별로 폴더가 나뉘어 있고, 각 폴더는 독립적으로 작동합니다.

## 먼저 할 일 : MCP 설치

이 에이전트들은 MCP 서버(Gmail · Notion · Google Sheets · Meta Ads 등)에 연결되어야 작동합니다.
설치는 아래 리포지토리를 먼저 따라 해주세요.

👉 **https://github.com/steveaimkt/marketing-os-mcp-setup**

## 폴더 구성

| Part | 주제 | 클립 수 |
|------|------|--------|
| part03 | 콘텐츠 파이프라인 | 3 |
| part04 | 리서치 자동화 | 5 |
| part05 | 콘텐츠 카피 | 5 |
| part06 | 광고 성과 | 6 |
| part07 | 데이터 분석 (GA4) | 3 |
| part08 | CRM 리텐션 | 3 |
| part09 | 확장 전략 | 3 |
| part10 | AX 팀 | 3 |

각 클립 폴더 안에는 다음이 들어 있습니다.

- `README.md` — 클립 개요
- `agent.md` — 에이전트 정의 (그대로 사용)
- `실습.md` — 실습 가이드
- `실습-N-M-*/` — 실습용 스킬 + 샘플 데이터 (가상 브랜드 데이터)

## 사용 방법

1. 위 MCP 설치 리포로 환경을 먼저 구성합니다.
2. 원하는 클립 폴더의 `agent.md` 내용을 본인 프로젝트의 에이전트로 등록합니다.
3. `실습.md` 를 따라 샘플 데이터로 한번 돌려본 뒤, 본인 데이터로 교체합니다.

> ⚠️ 샘플 데이터의 계정/숫자는 모두 가상(`act_000000000`, 가상 브랜드 "결로아/글로우랩")입니다.
> 실제 키는 포함되어 있지 않으니, MCP 설치 단계에서 본인 키를 `.env` 에 직접 넣어주세요.
