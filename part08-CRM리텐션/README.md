# Part 8. CRM·리텐션 — 고객 데이터를 매출로 바꾸는 3개 에이전트

> 이 Part에서 배우는 것: 흩어진 고객 데이터를 한 명으로 통합하고, CS 응대를 자동화하고, LTV를 분석해 다음 마케팅 행동을 정한다.

## 3개 에이전트

| # | 이름 | 역할 |
|---|---|---|
| 1 | **customer-data-unifier** | 자사몰·SNS·CS·광고 데이터의 고객 정보를 1명으로 통합 |
| 2 | **cs-responder** | 디스코드·이메일·SNS 문의 자동 분류·응대 (사람 승인 후 발송) |
| 3 | **ltv-analyzer** | LTV 세그먼트·재구매 패턴 분석 → 액션 추천 |
