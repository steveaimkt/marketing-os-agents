# 실습 — cs-responder

## Step 1. CS 위키 준비
노션 "CS 응대 위키" DB에 FAQ 20개 미리 입력 (배송·환불·사이즈·성분 등).

## Step 2. 시뮬레이션
```bash
claude --agent cs-responder "$(cat <<'EOF'
inbound:
  channel: "email"
  from: "customer@example.com"
  subject: "토너 사용 후 따가워요"
  body: "어제 받은 토너 쓰자마자 따가운데 환불 가능한가요?"
EOF
)"
# → 분류 🟡 + 답변 초안 + 디스코드 #marketing-approvals 승인 요청
```

## Step 3. 승인 흐름
디스코드에서 ✅ → 메일 자동 발송 + 노션 응대 이력에 기록
