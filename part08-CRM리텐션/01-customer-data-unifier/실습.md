# 실습 — customer-data-unifier

## Step 1. 시트 export 준비
- 자사몰 주문 CSV → 시트 탭 "Orders"
- CS 문의 export → 시트 탭 "CS"
- SNS 멘션 → voc-analyzer 결과 자동 로드

## Step 2. 실행
```bash
claude --agent customer-data-unifier "월간 통합"
# → "고객 마스터 DB" 시트에 정규화·통합된 행
```

## Step 3. 1인 고객 360 보기
```bash
claude --agent customer-data-unifier "이메일 user@example.com 360 리포트"
# → outputs/.../customer-360-user.html
```
