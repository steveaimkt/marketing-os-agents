# 클립 2-1 실습 — ad-performance-checker

## Step 1. 임계치 설정 (.env)
```bash
THRESHOLD_ROAS_WARN=2.0
THRESHOLD_ROAS_BLOCK=1.0
THRESHOLD_CPA_WARN=30000
THRESHOLD_CPA_BLOCK=50000
THRESHOLD_CTR_WARN=0.01
THRESHOLD_DAILY_BUDGET_WARN=1.1
THRESHOLD_DAILY_BUDGET_BLOCK=1.3
```

## Step 2. 첫 실행 (수동)
```bash
claude --agent ad-performance-checker "지금 즉시 점검"
# → 위반 없으면 "✓ 모든 캠페인 정상" (디스코드 알림은 안 보냄)
# → 위반 있으면 #marketing-alerts 채널에 alert embed
```

## Step 3. 매시간 자동 (Part 10에서)
```bash
# launchd 등록 미리보기
crontab -e
0 * * * * /usr/local/bin/claude --agent ad-performance-checker "정시 점검"
```

## 시뮬레이션
임계치를 일부러 높여서 위반을 강제 트리거:
```bash
THRESHOLD_ROAS_WARN=10.0 claude --agent ad-performance-checker "테스트"
# → 모든 캠페인이 위반으로 잡힘 → 알림 포맷 확인
```
