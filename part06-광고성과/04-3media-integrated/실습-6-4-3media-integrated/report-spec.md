# 3매체 통합 리포트 — 고도화 사양 (2026-07-07 확정 · 표준)

> 6-4 통합 리포트의 **표준 산출 형식**. 목표 = 광고 효율 + 예산 최적화.
> "비교"에서 멈추지 말고 **"어디서 빼서 어디에 넣어라"**까지 답하는 리포트.
> 이후 3media 리포트는 이 사양대로 생성한다. 표준 템플릿: `sample-output/3media-report-upgraded.html`.

## 0. 매체 고정 컬러 (전 리포트 일관 적용 · 필수)

| 매체 | HEX | 색 | 비고 |
|---|---|---|---|
| Meta | `#1877F2` | 블루 | |
| Google | `#F59E0B` | 앰버/골드 | 초록 금지 (Naver와 충돌) |
| Naver | `#03C75A` | 네이버 그린 | |

상태색은 매체색과 별개 존에서만: `good #16a34a` · `bad #ef4444`. 상단에 색 범례 항상 노출.

## 1. 다지표 비교 매트릭스 (섹션 1 · 판단용)

ROAS 하나로 판단 금지. **9지표**를 매체별로 나란히, **행마다 최고=초록·최저=빨강** 하이라이트.

| 지표 | 산식 | 좋은 방향 |
|---|---|---|
| 지출 | Σspend | — |
| 매출 | Σrevenue | 높을수록 |
| ROAS | 매출/지출 | 높을수록 |
| CPC | 지출/클릭 | 낮을수록 |
| CAC(=CPA) | 지출/전환 | 낮을수록 |
| CTR | 클릭/노출 | 높을수록 |
| CVR | 전환/클릭 | 높을수록 |
| CPM | 지출/노출×1000 | 문맥 |
| AOV | 매출/전환 | 문맥(일관성 점검) |

각 행에 "판단 포인트" 1줄. 핵심 해석: **Meta = 저CPC·저CAC 저가획득 엔진(CVR 낮아도 효율 1위) / 검색 = 고CVR이나 CPC가 CAC를 끌어올림** → "싸게 많이(Meta)" vs "비싸도 확실히(브랜드 검색)" 분리 운영.

## 2. 예산 최적화 코어 4종 (인라인 SVG · 외부 라이브러리 없음)

세그먼트 단위로 분해 후 집계 (Meta=광고세트, Google=브랜드/일반/저품질 키워드군, Naver=브랜드/일반).

**② 효율–지출 4분면 맵** — x=지출·y=ROAS·크기=매출·색=매체. 구분선: x=평균지출, y=블렌디드ROAS, BEP=1.0.
- 사분면: 우상 STAR(사수) · 좌상 SCALE(증액) · 우하 CUT(삭감) · 좌하 PRUNE(정리).
- 핵심 판독: **STAR 공백 = 돈이 승자에게 안 감** → 우하에서 회수해 좌상으로.

**③ 효율 프론티어 (한계 ROAS 곡선)** — 세그먼트를 ROAS 내림차순, 막대 폭=지출·높이=ROAS, 누적. BEP 교차점 표시.
- 핵심 판독: **BEP 미만 예산 비중(%)** = 잘라낼 꼬리. (이번 데이터: 45.7%가 ROAS 0.54)

**④ 재배분 워터폴** — 현재매출 → −저효율삭감분 → +고효율증액분 → 제안매출. ROAS 변화 병기.
- 효과를 **원 단위 + ROAS 델타**로. "제안·실행 아님" 배지 + 보수적 추정 주석 필수.

**⑤ 고의도 vs 저의도 3매체 그룹막대** — 매체별 (브랜드·전환 / 일반·상단) 2막대. BEP선.
- 핵심 판독: 3매체 공통 = 고의도 ≥ 3.8, 저의도 ≤ 1.2 → 매체 교체가 아니라 각 매체 저의도 세그 정리.

(보조 옵션 3종: ⑥지출-매출 기여 갭 Pareto · ⑦품질점수×ROAS 버블(Google) · ⑧WoW 추세(Meta). 요청 시 풀 대시보드.)

## 3. 인사이트 · 재배분 · 액션

- 인사이트 4~5줄: 다지표에서 도출("왜"). STAR 공백 / 범인 소수 세그 / 매체 성격 / 브랜드-일반 착시.
- 재배분: 매체 배분(현재→제안 %) + **세그먼트 단위 지시**(증액/−70%/입찰−20%/유지) 2열.
- 액션 체크박스 5~6개, 마지막은 "2주 후 재측정 → 워터폴 추정 검증".

## 4. 안전 원칙 (유지)

- 매체 간 ROAS 직접 비교 주의 주석(어트리뷰션 모델 차이) 필수.
- 재배분은 **제안까지만**. 예산 변경은 사용자 승인 영역.
- 산출: `outputs/{날짜}/3media-report-{날짜}.html` (단일 파일·인라인 CSS/SVG·다크모드·모바일).
