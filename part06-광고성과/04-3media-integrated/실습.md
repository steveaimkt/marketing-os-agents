# 클립 1-4 실습 — 3media-integrated-reporter

```bash
# 1) 3개 매체 분석 먼저 (병렬)
claude --agent meta-ads-analyzer "지난 7일" &
claude --agent google-ads-analyzer "지난 7일" &
claude --agent naver-ads-analyzer "지난 7일" &
wait

# 2) 통합
claude --agent 3media-integrated-reporter "지난 7일 통합"
# → outputs/.../3media-report-{날짜}.html
# → 매체별 비교 표 + 다음 주 채널 분배 추천
```

## 핵심 결과
- **인사이트 5문장** — "왜 이 매체가 더 잘됐나" 사람이 보고 결정할 수 있는 정보
- **채널 분배 추천** — 현재 분배 vs 효율 기반 추천을 비교 표로
