# 클립 1-3 실습 — naver-ads-analyzer

## 사전 — 네이버 검색광고 API 키
```
https://manage.searchad.naver.com → 도구 → API 사용 관리
→ API_KEY + SECRET_KEY + CUSTOMER_ID 발급
```

## 실행
```bash
claude --agent naver-ads-analyzer "지난 7일 네이버 검색광고 분석"
```

## 핵심 분석 결과
- **브랜드 키워드 ROAS**: 일반적으로 매우 높음 (10+)
- **일반 키워드 ROAS**: 카테고리에 따라 1~5
- 둘을 섞어 보면 진짜 효율을 못 봄 → 항상 분리
