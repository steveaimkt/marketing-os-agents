# 클립 2-2 실습 — ab-test-analyzer

```bash
claude --agent ab-test-analyzer "활성 A/B 테스트 모두 점검"
# → 각 테스트의 우승 확률 + 권장 액션
```

## 결과 해석
- **우승 확률 ≥ 95%** → "B안 우승, 채택 권장"
- **무승부 (차이 ≤ 5%)** → "다른 차원 시도 권장"
- **표본 부족 (전환 < 50)** → "더 수집"

## 노션 A/B 라이브러리 활용
우승 패턴이 적재되면 Part 5 `ad-copy-ab` 변형 생성 시 자동 가중치 부여.
